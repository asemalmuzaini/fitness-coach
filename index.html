Put it on your iPhone

Important: iPhone only offers “Add to Home Screen” when a page is opened in Safari from a web link — not when opened from the Files app. Two ways to get the app icon:

A) Quick host (one-time, on home Wi-Fi):

  1. On a computer, go to app.netlify.com/drop
  2. Drag this file in → you get a web link (URL).
  3. Open that link in Safari on your iPhone.
  4. Share (□↑) → Add to Home Screen → Add. Now it works offline.

B) No hosting: just open the file from the Files app each time. It still runs fully offline and saves your log — it just won’t have a home-screen icon.
