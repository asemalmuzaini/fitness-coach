What is RPE?

RPE = how hard it felt, 1–10. Use the talk test:

3–4Easy — you could chat all day
5–6Working — full sentences still OK
7Hard — short phrases only. This is the ceiling.
8+Too hard for this program — back off

For weights: RPE 6–7 means "I could have done 2–3 more reps".
